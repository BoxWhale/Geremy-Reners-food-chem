# Geremy Rener's Food Chemistry

A web programming project exploring the intersection of culinary arts and chemistry. We make food with science and passion!

## About

Welcome to Geremy Rener's Food Chemistry website. This project combines culinary arts with scientific principles to create extraordinary dishes. Our approach transforms cooking into an art form backed by chemistry and innovation.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling between sections
- **Recipe Showcase**: Featured recipes with molecular gastronomy techniques
- **Contact Form**: Get in touch with us directly through the website
- **Modern UI/UX**: Clean, professional design with engaging animations

## Technologies Used

- **HTML5**: Semantic markup for content structure
- **CSS3**: Modern styling with flexbox, grid, and animations
- **JavaScript**: Interactive features and dynamic content loading
- **No framework dependencies**: Pure vanilla JavaScript for maximum performance

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, but recommended for full functionality)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BoxWhale/Geremy-Reners-food-chem.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Geremy-Reners-food-chem
   ```

3. Open `index.html` in your web browser, or use a local server:
   
   **Using Python:**
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

   **Using Node.js (http-server):**
   ```bash
   npx http-server
   ```
